# Revora Landing Page

Landing page de captura para live gratuita sobre IA em atendimento e vendas.

## Setup

1. Clone o repositório
2. Abra `index.html` no navegador
3. Substitua `SEU_LINK_WHATSAPP` pelo link real do grupo

## Deploy

Conectado ao Vercel. Qualquer push para `main` faz deploy automático.

## Customização

- Cores em `index.html` (search por `#C0392B`, `#25D366`, etc)
- Copy e conteúdo na seção HTML
- Responsivo para mobile/tablet
